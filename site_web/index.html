Création des VMs

  1. Installez VirtualBox sur votre machine locale.
  2. Créez 5 machines virtuelles avec les systèmes d'exploitation et configurations suivantes :
    • 1 VM avec Windows pour client
    • 1 VM avec pfSense pour le pare-feu
    • 1 VM avec Linux pour client
    • 1 VM avec Linux pour le serveur web
    • 1 VM avec Linux pour le serveur de sauvegarde
  3. Configurez les VMs avec les paramètres appropriés pour la RAM, le stockage, et les cartes réseau.

Configuration réseau

  1. Configurez un réseau interne pour les VMs dans VirtualBox.
  2. Attribuez des adresses IP statiques aux VMs :
    • VM pfSense : WAN et LAN
    • VM client Linux : adresse IP statique
    • VM client Windows : adresse IP statique
    • VM serveur web : adresse IP statique
    • VM serveur de sauvegarde : adresse IP statique
  3. Configurez les règles de pare-feu dans pfSense pour autoriser le trafic entre les VMs et vers l'extérieur.

Configuration de la sauvegarde

  1. Installez un logiciel de sauvegarde (rsync) sur la VM dédiée à la sauvegarde.
  2. Configurez la sauvegarde pour copier les fichiers du serveur web vers la VM de sauvegarde à intervalles réguliers (par exemple, toutes les nuits).
  3. Testez le processus de sauvegarde en exécutant une sauvegarde manuelle et en vérifiant que les fichiers ont été correctement copiés sur la VM de sauvegarde.
  4. Configurez la restauration des fichiers sauvegardés en cas de besoin.

Bonnes pratiques

  1. Sauvegarde et restauration : Assurez-vous que les sauvegardes sont effectuées régulièrement et qu'elles peuvent être restaurées en cas de problème.
  2. Monitoring et alerting : Mettez en place un système de surveillance pour vérifier l'état de vos serveurs et services, et configurez des alertes pour être informé en cas de problème.
  3. Sécurité : Appliquez les mises à jour de sécurité, utilisez des mots de passe forts, et configurez les pare-feu pour limiter les risques de failles de sécurité.
  4. Documentation : Documentez votre infrastructure, les configurations et les procédures pour faciliter la maintenance et la résolution des problèmes.
